PROCESSOR MODULE

MX 200

MX207 / MX213 / MX220 + ColdClimate Variants

Processor modules of the MX200 series combine outstanding environmental obustness with very compact dimensions. Thanks to an integrated power supply unit, which can also provide the I/O module power supply and integrated communication interfaces, complete basic system can be implemented on only 2 module widths.
